How do I implement Sign in with ∞ in my Shortcuts?

Add a dictionary with the data you want as keys; Name for the username; E-mail if you want the e-mail address, Avatar to get the user’s avatar, add “ShortcutName” for the Shortcut name, but this is optional. When you do not add this, EasyAuth can not be used.

What is the Sign in with ∞ API (∞API)

This is an file based API, where you can get info about the Shortcut. Please check the README.html in /Shortcuts/∞API/
